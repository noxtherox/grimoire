## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-pick-vault-folder`
- `allow-restore-vault-folder`
- `allow-clear-vault-folder`
- `allow-pick-external-notes`
- `allow-pick-files`
- `allow-open-file`
